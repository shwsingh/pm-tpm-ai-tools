Bug Triage Agent + Blog Launch

Agent

06

Agent Workflow — 3-Stage Pipeline

Orchestration

Ingest → Triage → Escalation Handler. Pure functions, stage outputs feed next stage via session state. Draft incident summaries and on-call messages.

Code orchestrates Claude — the deliberate contrast to Day 12's agent loop.
07

Status Report Skill

Skill

Structured form → exec report → 5-dimension quality eval. Status color derived from content, not self-reported. Catches optimism bias.

  • Rubric: Ask Specificity, Status Accuracy, Impact Clarity, Next Week Concreteness, Exec Readability
  • Each criterion: ❌ wrong example + ✅ right example
08

Knowledge Base

RAG Foundation

Upload .txt, .md, .docx, .csv. Chunked and keyword-searched. No vector DB needed — chunking strategy matters more than the scorer at this scale.

Phase 03 · Days 9–12

LLM & Orchestration

First Claude calls · Dependency graph · AgentHarness · Multi-agent loop

09

Feedback Agent + First Claude Calls

LLM

Heuristic triage swapped for Claude — one function change, zero pipeline changes. Feedback Agent: per-item sentiment, themes, severity, action + aggregate TPM next steps.

The Day 5 contract meant the LLM swap touched nothing downstream.
10

Dependency Agent

LLM

Structured form for cross-team dependencies. Claude reasons over the full graph — critical path, cascading risks, TPM actions. One call, not one per dependency.

Cascade detection requires seeing the full graph at once.
11

AgentHarness + Evaluation Framework

Infrastructure

All Claude calls unified through a single AgentHarness. Claude-as-judge scores any output against skill spec rules — criterion by criterion, pass/fail with reason.

  • Retry on JSON parse failure only — API errors surface immediately
  • Two-call eval pattern: agent produces, evaluator scores
  • Token dashboard: cumulative cost visible across the session
12

Multi-Agent Orchestrator

Multi-Agent

Free-text TPM request → agent loop → Claude picks tools → executes → synthesizes exec briefing. Every tool call visible: input, result, token count.

  • 4 tools: triage_bug, analyze_feedback, analyze_dependencies, search_knowledge_base
  • Max 5 iterations. Tool errors continue the loop, don't crash it
First surface where Claude orchestrates itself — not the code.
